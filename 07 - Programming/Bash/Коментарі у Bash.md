# Коментарі у Bash

Щоб позначити коментар у [[Bash]] використовується символ `#` (окрім першого рядка [[shebang]]).

> [!example]
> ```bash
> # це коментар
> var="value" # це теж коментар
> ```

> [!warning] Важливо
>  Коментарі ігноруються оболонкою під час виконання.
