> [!info] Змінити каталог

```bash
cd [шлях_до_директорії]
```

## Опції 

- `cd` — перейти до [[Домашній каталог користувача]]
- `cd -` — перейти до попередньої робочої директорії
- `cd ~user_name` — перейти до директорії користувача з `user_name`
