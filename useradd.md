> [!info] Створити користувача

## Синтаксис

```bash
useradd [опції] username
```

## Опції

- `-m` - створити домашній каталог
- `-d /path` - вказати домашній каталог
- `-g group` - основна група
- `-G group1,group2` - додаткові групи
