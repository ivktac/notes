> [!tldr]
> **SELECT** основна команда [[SQL]] для отримання (вибірки) даних з [[таблиця|таблиць]] бази даних.

```sql
SELECT column1, column2, ...
FROM table_name;
```

## Навіщо?

- переглянути, що зберігається у таблиці
- створити звіт даних
- проаналізувати інформацію
- отримати дані для додатків
