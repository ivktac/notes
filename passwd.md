> [!info] Змінити пароль

> [!warning] За допомогою [[Делагування повноважень|sudo]] можна змінити пароль для будь-якого користувача!!!

## Синтаксис

```bash
passwd [опції] [username]
```

## Опції

- `-d` - видалити пароль
- `-l` - заблокувати
- `-u` - розблокувати
- `-S` - статус
