| Символ | Опис                                                             |
| ------ | ---------------------------------------------------------------- |
| `\b`   | Поверення на крок                                                |
| `\f`   | Зміна сторінки                                                   |
| `\n`   | Зміна рядка (новий рядок)                                        |
| `\r`   | Повернення каретки                                               |
| `\t`   | Табуляція                                                        |
| `\ddd` | вісімкове значення $ddd$, де $ddd$ це 3 цифри від 0 до 7         |
| `\c`   | будь-який інший символ `c` (напр. `\\` для бекслеша, `\"` для ") |
