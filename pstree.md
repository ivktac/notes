> [!tldr]
> **pstree** — утиліта, що відображає дерево процесів.

```bash
pstree [опції] [pid або ім'я_користувача]
```

## Опції

- `-p` — показати PID
- `-n` — сортувати по PID
- `-h` — підсвітка поточного процесу з нащадками
- `-a` — розгорнутий вигляд

> [!tip] Для чого?
> Дозволяє аналізувати структуру запущених процесів.
