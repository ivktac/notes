> [!info]
> Розпаковує файли стиснені [[bzip2]]. Підтримує розширення: `.bz2`, `.bz`, `.tbz2`, `.tbz`

```bash
bunzip2 [options] [files]
```

## Опції

- `-c` - вивести результат на stdout
- `-f` - примусово перезаписати існуючі файли
- `-k` - не видаляти вхідні файли
