- Додати колонку: `ALTER TABLE table_name ADD column_name datatype;`
- Видалити колонку: `ALTER TABLE table_name DROP COLUMN column_name;`
- Змінити тип: `ALTER TABLE table_name MODIFY COLUMN column_name datatype;`
