| Оператор | Пояснення              | Приклад                        |
| -------- | ---------------------- | ------------------------------ |
| `<`      | менше ніж              | `NF < 10`                      |
| `<=`     | менже ніш або дорівнює | `NR <= 150`                    |
| `==`     | дорівнює               | `$1 == 'SomeString'`           |
| `!=`     | не дорівнює            | `$1 != 'NotExists'`            |
| `>=`     | більше або дорівнює    | `$2/$3 >= 0.5`                 |
| `>`      | більше ніж             | `$2 > 10`                      |
| `~`      | збігається з           | `$4 ~ /linux/` (або `"linux"`) |
| `!~`     | не збігається з        | `$5 !~ /awk/`                  |
