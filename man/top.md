> [!tldr]
> `top` — це утиліта для моніторингу [[Процес|процесів]], CPU, пам'яті та навантаження системі у реальному часі.

## Гарячі клавіші

- `1` — показати статистику по кожному ядру CPU
- `M` — відсортувати за споживання пам'яті
- `P` — відсортувати за CPU
- `k` — завершити процес (потрібно ввести `PID`)
- `q` — вийти

## Поля

Кожне поло показує на що процесор витрачає час (у відсотках `%`)

- `us` — виконання коду користувача без змінених nice (user space)
- `sy` — виконання процесів ядра (kernel/system space)
- `ni` — виконання коду користувача із зміненим nice
- `id` — простій процесора
- `wa` — очікування завершення задач вводу/виводу
- `hi` — обробка апаратних переривань
- `si` — обробка програмних переривань
- `st` — забраний від ВМ гіпервізором
